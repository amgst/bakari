# Bakari Capital Landing Page

## Overview
A minimalist landing page for Bakari Capital featuring elegant typography and a clean design aesthetic. Built with pure HTML and CSS.

## Project Architecture
- **index.html** - Main HTML structure
- **styles.css** - Styling with Google Fonts (Archivo Expanded, Cardo)
- Color palette: Beige (#e8e4dd), Yellow (#f4e96d), Black (#000000)

## Recent Changes
- **Nov 3, 2025**: Initial implementation created based on design mockup
  - Responsive layout with mobile breakpoints
  - Typography using Archivo Expanded and Cardo fonts
  - Three-section layout: header, hero, footer

## Features
- Fully responsive design
- Clean, minimalist aesthetic
- Custom typography with Google Fonts
- Mobile-optimized layout
